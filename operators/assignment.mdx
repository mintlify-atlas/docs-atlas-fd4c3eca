# Assignment Operators

Assignment operators in Dart are used to assign values to variables and perform operations simultaneously.

## Basic Assignment

The basic assignment operator `=` assigns a value to a variable.

```dart
int x = 10;
```

## Compound Assignment Operators

Compound assignment operators combine an arithmetic operation with assignment, providing a shorthand syntax.

## Complete Example

Here's a practical example showing how assignment operators work in sequence:

```dart
void main() {
  int x = 10;

  x += 5;    // x = x + 5  → x = 15
  x -= 3;    // x = x - 3  → x = 12
  x *= 2;    // x = x * 2  → x = 24
  x ~/= 4;   // x = x ~/ 4 → x = 6
  print('Valor de X: $x');

}
```

## Operator Breakdown

### Addition Assignment (`+=`)
Adds the right operand to the variable and assigns the result.

```dart
int x = 10;
x += 5;  // Equivalent to: x = x + 5
print(x);  // 15
```

### Subtraction Assignment (`-=`)
Subtracts the right operand from the variable and assigns the result.

```dart
int x = 15;
x -= 3;  // Equivalent to: x = x - 3
print(x);  // 12
```

### Multiplication Assignment (`*=`)
Multiplies the variable by the right operand and assigns the result.

```dart
int x = 12;
x *= 2;  // Equivalent to: x = x * 2
print(x);  // 24
```

### Division Assignment (`/=`)
Divides the variable by the right operand and assigns the result as a double.

```dart
double x = 24;
x /= 4;  // Equivalent to: x = x / 4
print(x);  // 6.0
```

### Integer Division Assignment (`~/=`)
Divides the variable by the right operand and assigns only the integer part.

```dart
int x = 24;
x ~/= 4;  // Equivalent to: x = x ~/ 4
print(x);  // 6
```

<Note>
  The `~/=` operator is unique to Dart and performs integer division with assignment.
</Note>

### Modulo Assignment (`%=`)
Calculates the remainder and assigns it to the variable.

```dart
int x = 10;
x %= 3;  // Equivalent to: x = x % 3
print(x);  // 1
```
