## Introduction

Every programming journey begins with a simple "Hello World" program. In Dart, this is incredibly straightforward and demonstrates the basic structure of a Dart application.

## The Hello World Program

Here's the complete Hello World program in Dart:

```dart
void main() {
  print('Hola Mundo!!!');
}
```

## Understanding the Code

### The `main()` Function

Every Dart application must have a `main()` function, which serves as the entry point of the program. When you run a Dart application, execution begins at the `main()` function.

<Info>
The `void` keyword indicates that the `main()` function doesn't return any value.
</Info>

### The `print()` Function

The `print()` function is used to output text to the console. It takes a string as an argument and displays it when the program runs.

```dart
print('Hola Mundo!!!');
```

<Tip>
You can use either single quotes (`'`) or double quotes (`"`) for strings in Dart. Both work the same way!
</Tip>

## Running the Program

To run this program:

1. Save the code in a file with a `.dart` extension (e.g., `hello_world.dart`)
2. Open your terminal
3. Navigate to the directory containing the file
4. Run: `dart hello_world.dart`

You'll see the output:
```
Hola Mundo!!!
```
