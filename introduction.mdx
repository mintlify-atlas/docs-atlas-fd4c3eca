## Your Learning Path

This documentation follows a progressive structure with 32 carefully ordered examples:

### Getting Started (Examples 1-4)
Start with the basics: your first program, variable naming conventions, data types, and constants.

```dart
void main() {
  print('Hola Mundo!!!');
}
```

### Operators (Examples 5-9)
Master arithmetic, assignment, comparison, logical, and ternary operators.

```dart
// Arithmetic operations
int suma = 10 + 5;
int multiplicacion = 4 * 3;

// Comparison operators
bool esMayor = 10 > 5;  // true
bool esIgual = 10 == 10; // true
```

### Control Flow (Examples 10-12)
Learn decision-making with if/else statements, type conversions, and switch cases.

```dart
int calificacion = 90;

if (calificacion >= 60) {
  print('Acredito');
} else {
  print('No acredito');
}
```

### Loops (Examples 13-16)
Explore different iteration patterns including for, while, and do-while loops.

```dart
// Traditional for loop
for (int i = 1; i <= 5; i++) {
  print('Iteración $i');
}

// Custom increment
for (var i = 0; i <= 10; i += 2) {
  print('Iteración $i');
}
```

### Functions (Examples 17-22)
From basic functions to advanced parameter handling and arrow syntax.

```dart
// Basic function without return
void saludar(){
  print('Hola, estoy usando Funciones Básicas sin Retorno');
}

// Function with parameters
void imprimirNombre(String nombre){
  print('Hola, $nombre');
}

// Arrow function
int duplicar(int numero) => numero * 2;
```

### Collections (Examples 23-32)
Master Lists and Maps with practical operations: add, remove, search, and iterate.

```dart
// Lists
List<int> numeros = [10, 20, 30];
numeros.add(40);
numeros.addAll([50, 60]);
print('Tamaño: ${numeros.length}');
print('Primer elemento: ${numeros.first}');

// Maps (Dictionaries)
Map<String, int> edades = {
  'Juan': 17,
  'María': 16,
  'Carlos': 18
};
print(edades['Juan']); // 17
```
